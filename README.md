
# Potato Leaf Disease Detection Using CNN

This project uses a Convolutional Neural Network (CNN) to detect potato leaf diseases from leaf images.

## Classes
- Early Blight
- Late Blight
- Healthy

## Dataset
PlantVillage Potato Dataset  
https://www.kaggle.com/datasets/aarishasifkhan/plantvillage-potato-disease-dataset

## Technologies Used
- Python
- TensorFlow / Keras
- Convolutional Neural Network (CNN)
- Google Colab

## Results
Training Accuracy: ~98%  
Validation Accuracy: ~99%
